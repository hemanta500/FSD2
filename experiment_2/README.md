# Experiment 2

## Learning Outcomes

1. CompoBuilding with Blocks: Learn how to make a complete screen by putting together smaller parts like buttons, checkboxes, and star ratings.

2. Saving Information: Learn how to use "state" so the app remembers what the user clicks or types while they are using it.

3. Making it Look Good: Use a tool called Material-UI to make your app look clean and professional without having to design everything from scratch.

4. Filling Out Forms: Practice making boxes where people can type information and learn how to save that info correctly.

5. Moving Around: Understand how to build an app that lets users jump between different pages smoothly and quickly.
